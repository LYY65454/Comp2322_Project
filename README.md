# Comp2322 Multi-thread Web Server

This project implements a multi-threaded Web server using Python socket programming. It supports HTTP GET and HEAD methods, multiple status codes, and concurrent client connections.

## Requirements:
- Python 3.x

## How to run:

1. Run the server:
   python server.py

2. Open browser:
   http://127.0.0.1:<PORT>/

The port number is displayed in the terminal output.

## How to stop the server

Press `Ctrl + C` in the terminal to stop the server.
